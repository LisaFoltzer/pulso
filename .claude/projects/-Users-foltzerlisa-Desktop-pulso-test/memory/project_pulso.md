Pulso is an automated process audit SaaS for PME (SMBs). Core value prop: connects to a company's emails, meetings, and calls, automatically detects all business processes, maps them, and provides a daily report with AI recommendations.

**Two main sections:**
1. **Analysis** — timeline of events, flow map of exchanges between people/teams
2. **Report/Bilan** — dashboard, health scores, AI insights, recommendations

**Tech stack:** Next.js, TypeScript, Tailwind CSS, Supabase, Claude API for AI features.

**Design system:** White background #F8FAFC, dark sidebar (#0F172A), DM Sans font, professional style inspired by Linear/Notion.

**Current state:** Working with mock/fictitious data. No backend integration yet.

**How to apply:** Build features directly when asked, no need to confirm approach. Use mock data. Follow the established design system and component patterns.
